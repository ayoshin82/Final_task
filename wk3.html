Git

  • git branch. Это менеджер веток, который позволяет перечислять, создавать новые, удалять и переименовывать их.
  • git checkout. Используется для переключения веток и выгрузки их содержимого в рабочую директорию.
  • git push. Используется для установления связи с удалённым репозиторием, вычисления локальных изменений, отсутствующих в нём, и их передачи в репозиторий.
  • git rm. Используется для удаления файлов из индекса и рабочей директории.
  • git add используется как отдельная команда, она переносит ожидающие изменения из рабочего каталога в раздел проиндексированных файлов.
